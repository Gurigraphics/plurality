About

Plurality is a software that combines Music Editor, Code editor, Story Editor, Game Engine and Project Management. All in one place.

Plurality adapts to your project, team and workflow. All that is related to your project is visible and updated in real time. No need to configure anything, and everyone receives the information instantly.

Plurality is perfectly synchronized with all your devices, wherever you are.


Develop now is a game

Plurality will not drown you in resources that you do not use.
To activate Tools Extras you need conquest them.

In the tutorials the users learn to develop their own code blocks to create your own Engines. The higher the user level, the more resources are at your disposal. This is gamification dev
